# simple-camera by katahiromz

## 概要

React+Vita製のシンプルなカメラアプリです。PWAとAndroidに対応しております。

  <img src="img/screenshot.png" alt="[スクリーンショット]" height="192" />

## 機能

- 写真撮影と録画
- 二本指によるズーム
- ズーム時に指で表示位置の移動が可能
- 前面カメラ・背面カメラの切り替えが可能

## 技術仕様

- **言語**: TypeScript
- **フレームワーク**: React 19
- **ビルドツール**: Vite 7
- **録画**: MediaRecorder API with Canvas Stream Capture
- **国際化**: i18next (日本語・英語対応)
- **アイコン**: lucide-react
